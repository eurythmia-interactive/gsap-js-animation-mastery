Lesson 14 Part 2: Advanced

ScrollTrigger Callbacks

Use ScrollTrigger callbacks to trigger actions at specific scroll positions.

Demo - Progress Indicator 0%

The Code

JavaScript

JavaScript 
import { gsap } from 'gsap';
import { ScrollTrigger } from 'gsap/ScrollTrigger';

gsap.registerPlugin(ScrollTrigger);

const indicator = document.querySelector('#indicator');
const progressFill = document.querySelector('#progressFill');
const progressText = document.querySelector('#progressText');

// Bounce the indicator on enter
gsap.fromTo(indicator, 
  { scale: 0, opacity: 0 },
  {
    scale: 1,
    opacity: 1,
    duration: 0.5,
    ease: 'back.out',
    scrollTrigger: {
      trigger: '#demo',
      start: 'top 80%',
      once: true
    }
  }
);

// Update progress as you scroll
ScrollTrigger.create({
  trigger: '#demo',
  start: 'top bottom',
  end: 'bottom top',
  onUpdate: function(self) {
    const progress = Math.round(self.progress * 100);
    progressFill.style.width = progress + '%';
    progressText.textContent = progress + '%';
  }
});

How It Works

ScrollTrigger provides lifecycle callbacks: onEnter, onLeave, onUpdate, and more.

Available Callbacks

  • onEnter - When scroll enters the trigger zone (forward)
  • onLeave - When scroll leaves the trigger zone
  • onUpdate - Fires every frame during scroll
  • onToggle - Fires when visibility toggles

self.progress

In onUpdate, self.progress is a value from 0 to 1 representing scroll position within the trigger range.
