Lesson 01 Part 2: Advanced

gsap.quickTo() & quickSetter()

Supercharge animations that need to update frequently, like mouse followers or scroll-linked effects.

Demo

The Code

HTML

HTML 
<div class="demo-container" id="demo">
  <div class="demo-box" id="follower"></div>
</div>

CSS

CSS 
.demo-box {
  width: 80px;
  height: 80px;
  border-radius: 12px;
  position: absolute;
  left: 0;
  top: 0;
}

#follower {
  background: linear-gradient(135deg, #00d4ff, #8b5cf6);
  z-index: 2;
}

JavaScript

JavaScript 
import { gsap } from 'gsap';

const follower = document.querySelector('#follower');
const demo = document.querySelector('#demo');

// Create quickTo functions - these are OPTIMIZED for frequent updates
const moveX = gsap.quickTo(follower, 'x', { duration: 0.5, ease: 'power2.out' });
const moveY = gsap.quickTo(follower, 'y', { duration: 0.5, ease: 'power2.out' });

// Mouse move handler - calls the quickTo function
demo.addEventListener('mousemove', (e) => {
  const rect = demo.getBoundingClientRect();
  moveX(e.clientX - rect.left - 40);
  moveY(e.clientY - rect.top - 40);
});

// Reset button
document.querySelector('#resetBtn').addEventListener('click', () => {
  moveX(0);
  moveY(0);
});

How It Works

gsap.quickTo() creates a highly optimized function that you can call repeatedly. Unlike regular gsap.to() calls which create new tweens each time, quickTo() caches the animation logic for instant execution.

The Problem

If you call gsap.to(element, {x: 100}) inside a mouse move handler that fires 60 times per second, you're creating 60 new tweens! This kills performance.

The Solution

gsap.quickTo() pre-calculates everything and returns a simple function you can call anytime. It internally uses gsap.quickSetter() for maximum performance.

quickSetter vs quickTo

  • quickSetter - Sets a raw value directly (no animation). Use for position updates that don't need easing.
  • quickTo - Animates to the value with easing. Use when you want smooth motion.

When to Use

QuickTo is perfect for: mouse followers, scroll-linked animations, parallax effects, and any animation that needs to update dozens of times per second.
