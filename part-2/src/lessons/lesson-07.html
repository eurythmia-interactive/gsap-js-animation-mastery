ScrollTrigger Pin & Snap

Pin elements in place while scrolling, with optional snap points.

The Code

JavaScript

JavaScript 
import { gsap } from 'gsap';
import { ScrollTrigger } from 'gsap/ScrollTrigger';

gsap.registerPlugin(ScrollTrigger);

// Simple pin animation
gsap.fromTo('#pinContent', 
  { opacity: 0, y: 50 },
  {
    opacity: 1,
    y: 0,
    duration: 1,
    scrollTrigger: {
      trigger: '#pinDemo',
      start: 'top top',
      end: pin: true,
      scrub: 1,
      anticipatePin: 1
    }
  }
);

How It Works

pin: true tells ScrollTrigger to fix the trigger element in place while the animation plays, creating sticky-section effects.

Pin Options

  • pin: true - Pin the trigger element
  • pinSpacing: true - Add space after the pin (default)
  • anticipatePin: 1 - Prevents jump on fast scroll

Snap with ScrollTrigger

GSAP ScrollTrigger has built-in snap support. Enable it in ScrollTrigger:

scrollTrigger: {
  snap: {
    snapTo: 1 / 5,  // Snap to one of 5 equal divisions
    duration: 0.5,
    delay: 0.1
  }
}
