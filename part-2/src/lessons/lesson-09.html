Lesson 09 Part 2: Advanced

SVG Transforms

Animate SVG elements with transforms for scaling, rotation, and more.

Demo - SVG Shape Animation

The Code

HTML

HTML 
<svg viewBox="0 0 100 100" width="200" height="200">
  <path id="morphPath" d="M 50,10 A 40,40 0 1,1 49.99,10" />
</svg>

JavaScript

JavaScript 
import { gsap } from 'gsap';

const path = document.querySelector('#morphPath');

let isExpanded = true;

document.querySelector('#morphBtn').addEventListener('click', () => {
  if (isExpanded) {
    gsap.to(path, {
      scaleX: 1.3,
      scaleY: 0.7,
      transformOrigin: '50% 50%',
      duration: 1,
      ease: 'power2.inOut'
    });
  } else {
    gsap.to(path, {
      scaleX: 1,
      scaleY: 1,
      duration: 1,
      ease: 'power2.inOut'
    });
  }
  isExpanded = !isExpanded;
});

How It Works

SVG elements can be animated just like HTML elements using GSAP. Use transformOrigin to control the pivot point.

SVG Transform Tips

  • transformOrigin: '50% 50%' - Center of the SVG
  • transformOrigin: '0% 0%' - Top-left corner
  • Use scaleX, scaleY for independent scaling

GSAP Club Bonus

MorphSVG (Club bonus) can smoothly morph between completely different path shapes, even with different point counts!
