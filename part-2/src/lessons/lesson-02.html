JavaScript

JavaScript 
import { gsap } from 'gsap';
import { Flip } from 'gsap/Flip';

gsap.registerPlugin(Flip);

const cards = document.querySelectorAll('.demo-card');

// Click to toggle expanded state
cards.forEach(card => {
  card.addEventListener('click', () => {
    const state = Flip.getState(cards);
    
    card.classList.toggle('expanded');
    
    Flip.from(state, {
      duration: 0.5,
      ease: 'power2.inOut',
      stagger: 0.05,
      absolute: true
    });
  });
});

How It Works

The Flip plugin captures the current state of elements (position, size, rotation) before a change, then animates them from that state to their new state.

FLIP = First, Last, Invert, Play

First: Record the initial positions
Last: Make the DOM change
Invert: Calculate the difference
Play: Animate from start to end

Key Methods

  • Flip.getState(targets) - Captures current positions of elements
  • Flip.from(state) - Animates from captured state to current state
  • Flip.to(state) - Animates from current state to captured state

The "absolute" Option

Setting absolute: true uses absolute positioning during the animation to prevent elements from jumping around as siblings change size.
