Lesson 10 Part 2: Advanced

Draggable Rotation

Use Draggable to create rotation-based interactions.

Demo - Drag to Rotate

The Code

JavaScript

JavaScript 
import { gsap } from 'gsap';
import { Draggable } from 'gsap/Draggable';

gsap.registerPlugin(Draggable);

const box = document.querySelector('#throwBox');
const demo = document.querySelector('#demo');

Draggable.create(box, {
  type: 'x,y',
  bounds: demo,
  onDrag: function() {
    gsap.to(box, {
      rotation: '-=2',
      duration: 0.1,
      overwrite: true
    });
  },
  onDragEnd: function() {
    gsap.to(box, {
      rotation: '-=30',
      duration: 0.5,
      ease: 'elastic.out(1, 0.5)'
    });
  }
});

document.querySelector('#resetBtn').addEventListener('click', () => {
  gsap.to(box, {
    rotation: 0,
    duration: 0.5,
    ease: 'power2.out'
  });
});

How It Works

Draggable tracks the movement of elements and provides callbacks for drag events. Combine with GSAP for smooth animations.

Key Callbacks

  • onDragStart - Called when drag begins
  • onDrag - Called every frame during drag
  • onDragEnd - Called when drag ends

GSAP Club Bonus

InertiaPlugin (Club bonus) adds physics-based momentum - elements "throw" with realistic coasting!
