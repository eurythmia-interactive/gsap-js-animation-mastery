Lesson 06 Part 2: Advanced

MotionPath Plugin

Animate elements along SVG paths with precise control over position and rotation.

Demo - Motion Along Path

The Code

HTML

HTML 
<svg class="motion-path-svg" viewBox="0 0 400 200">
  <path class="motion-path-line" d="M 20,100 C 100,20 300,180 380,100" />
</svg>
<div id="dot"></div>

JavaScript

JavaScript 
import { gsap } from 'gsap';
import { MotionPathPlugin } from 'gsap/MotionPathPlugin';

gsap.registerPlugin(MotionPathPlugin);

const dot = document.querySelector('#dot');
const path = document.querySelector('.motion-path-line');

let forward = true;

const tl = gsap.timeline({ paused: true, repeat: -1, yoyo: true });

tl.to(dot, {
  motionPath: {
    path: path,
    align: path,
    alignOrigin: [0.5, 0.5],
    autoRotate: true
  },
  duration: 3,
  ease: 'none'
});

document.querySelector('#playBtn').addEventListener('click', () => tl.play());
document.querySelector('#reverseBtn').addEventListener('click', () => tl.reverse());
document.querySelector('#toggleDirection').addEventListener('click', () => {
  forward = !forward;
  tl.timeScale(forward ? 1 : -1);
});

How It Works

MotionPathPlugin allows you to animate any element along an SVG path. The element's rotation automatically follows the path tangent.

Key Properties

  • path - The SVG path element or path data string
  • align - Align the element to this path
  • alignOrigin - Which point of the element aligns to the path (0.5, 0.5 = center)
  • autoRotate - Automatically rotate to follow the path

offsetX and offsetY

Use offsetX and offsetY to shift the element perpendicular to the path.
