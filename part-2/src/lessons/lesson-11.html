Lesson 11 Part 2: Advanced

SVG Animation

Animate SVG elements including stroke-dasharray for draw-on effects.

Demo - Draw On Effect

The Code

HTML

HTML 
<svg viewBox="0 0 200 200">
  <circle class="svg-circle" 
     cx="100" cy="100" r="80"
     fill="none" stroke="#00d4ff"
     stroke-width="4"
     stroke-dasharray="503"
     stroke-dashoffset="503" />
</svg>

JavaScript

JavaScript 
import { gsap } from 'gsap';

const circle = document.querySelector('.svg-circle');
const circumference = 2 * Math.PI * 80; // ~503

// Set initial state - fully hidden
gsap.set(circle, { strokeDashoffset: circumference });

document.querySelector('#drawBtn').addEventListener('click', () => {
  gsap.to(circle, {
    strokeDashoffset: 0,
    duration: 2,
    ease: 'power2.inOut'
  });
});

document.querySelector('#resetBtn').addEventListener('click', () => {
  gsap.to(circle, {
    strokeDashoffset: circumference,
    duration: 0.5,
    ease: 'power2.in'
  });
});

How It Works

SVG stroke animation uses stroke-dasharray and stroke-dashoffset to create draw-on effects.

The Math

stroke-dasharray = Length of dashes
stroke-dashoffset = How much dash is hidden

When dashoffset equals dasharray, nothing is visible. When offset is 0, the full dash shows.

Calculating Circumference

For a circle: 2 * Math.PI * radius. This gives you the exact dasharray value needed.
