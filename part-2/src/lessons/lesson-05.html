Parallax Scrolling

Create depth illusion by moving layers at different speeds during scroll.

JavaScript

JavaScript 
import { gsap } from 'gsap';
import { ScrollTrigger } from 'gsap/ScrollTrigger';

gsap.registerPlugin(ScrollTrigger);

// Background - slowest (0.2 speed)
gsap.to('.parallax-bg', {
  y: -100,
  ease: 'none',
  scrollTrigger: {
    trigger: '#demo',
    start: 'top bottom',
    end: 'bottom top',
    scrub: true
  }
});

// Shapes - medium speed (0.5 speed)
gsap.to('.parallax-shapes', {
  y: -200,
  ease: 'none',
  scrollTrigger: {
    trigger: '#demo',
    start: 'top bottom',
    end: 'bottom top',
    scrub: true
  }
});

// Content - opposite direction for extra depth
gsap.to('.parallax-content', {
  y: 50,
  ease: 'none',
  scrollTrigger: {
    trigger: '#demo',
    start: 'top bottom',
    end: 'bottom top',
    scrub: true
  }
});

How It Works

Parallax creates depth by moving elements at different speeds during scroll. Background elements move slower than foreground elements.

The Key Principle

Different y values moved over the same scroll distance create the parallax effect. Smaller movements = background, larger movements = foreground.

Speed Factors

  • Background: 0.2 - 0.3 (very slow, creates depth)
  • Mid-ground: 0.5 - 0.7 (medium speed)
  • Foreground: 1.0 - 1.5 (faster or even opposite)

scrub: true

Using scrub: true (or a number for smoothing) ties the animation directly to scroll position, creating a smooth parallax effect.
