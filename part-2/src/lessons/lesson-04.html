Horizontal Scroll

Create horizontal scrolling sections using ScrollTrigger with pin and scrub.

Demo - Scroll Down
Scroll to explore →

The Code

HTML

HTML 
<div class="demo-container" id="demo">
  <div class="horizontal-wrapper">
    <div class="horizontal-panel">Panel 1</div>
    <div class="horizontal-panel">Panel 2</div>
    <div class="horizontal-panel">Panel 3</div>
  </div>
</div>

CSS

CSS 
.horizontal-wrapper {
  display: flex;
  width: 300%; /* 3 panels = 300% */
}

.horizontal-panel {
  flex: 0 0 100%;
  height: 100%;
  display: flex;
  align-items: center;
  justify-content: center;
  font-size: 2rem;
  font-weight: 700;
}

JavaScript

JavaScript 
import { gsap } from 'gsap';
import { ScrollTrigger } from 'gsap/ScrollTrigger';

gsap.registerPlugin(ScrollTrigger);

const wrapper = document.querySelector('.horizontal-wrapper');
const panels = document.querySelectorAll('.horizontal-panel');

// Set initial state
gsap.set(wrapper, { x: 0 });

// Create horizontal scroll animation
const tl = gsap.timeline({
  scrollTrigger: {
    trigger: '#demo',
    start: 'top top',
    end: '+=200%', /* 2x the panel width */
    pin: true,
    scrub: 1,
    anticipatePin: 1
  }
});

tl.to(wrapper, {
  x: -window.innerWidth * 2,
  ease: 'none'
});

// Animate each panel content
panels.forEach((panel, i) => {
  gsap.from(panel, {
    opacity: 0,
    y: 50,
    duration: 0.5,
    scrollTrigger: {
      trigger: panel,
      containerAnimation: tl,
      start: 'left center',
      toggleActions: 'play none none reverse'
    }
  });
});

How It Works

Horizontal scrolling in GSAP uses ScrollTrigger with pin: true to keep the container in place while we animate its x position based on scroll progress.

The Key Setup

  • pin: true - Pins the trigger element during the animation
  • end: "+=200%" - Makes the scroll distance 2x the viewport width
  • scrub: 1 - Ties the animation to scroll position with 1 second smoothing

containerAnimation

When animating children of a ScrollTrigger container, use containerAnimation to tell GSAP which timeline to sync with.
