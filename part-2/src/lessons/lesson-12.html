JavaScript

JavaScript 
import { gsap } from 'gsap';

const cards = document.querySelectorAll('.batch-card');

// Set initial state
gsap.set(cards, { opacity: 0, y: 50 });

// Create batch animation
const batch = gsap.batch(cards, {
  duration: 0.5,
  ease: 'power2.out',
  stagger: 0.1,
  onUpdate: function(self) {
    // self.items = elements in this batch
    // self.events = batch event (start, end, enter)
  }
});

document.querySelector('#batchBtn').addEventListener('click', () => {
  batch.enter();
});

document.querySelector('#reverseBtn').addEventListener('click', () => {
  batch.reverse();
});

How It Works

gsap.batch() creates a smart batch animation that handles elements in groups, reducing the number of callbacks and improving performance for large lists.

Batch Methods

  • batch.enter() - Animate elements that match the selector
  • batch.exit() - Reverse animation for matching elements
  • batch.reverse() - Toggle direction

Performance

Batch is especially useful when animating 50+ elements. It groups updates to reduce layout thrashing.
