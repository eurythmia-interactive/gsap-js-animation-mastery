The Code

HTML

HTML 
<div class="demo-container">
  <div class="batch-card">Card 1</div>
  <div class="batch-card">Card 2</div>
  <div class="batch-card">Card 3</div>
</div>

JavaScript

JavaScript 
import { gsap } from 'gsap';

const cards1 = document.querySelectorAll('.batch-card');
const cards2 = document.querySelectorAll('.batch-card-2');

gsap.set(cards1, { opacity: 0, y: 50 });
gsap.set(cards2, { opacity: 0, y: 50 });

document.querySelector('#batchBtn1').addEventListener('click', () => {
  gsap.to(cards1, {
    opacity: 1, y: 0,
    duration: 0.5, ease: 'power2.out',
    stagger: 0.1
  });
});

document.querySelector('#reverseBtn1').addEventListener('click', () => {
  gsap.to(cards1, {
    opacity: 0, y: 50,
    duration: 0.5, ease: 'power2.out',
    stagger: 0.1
  });
});

document.querySelector('#batchBtn2').addEventListener('click', () => {
  gsap.to(cards2, {
    opacity: 1, y: 0,
    duration: 0.5, ease: 'power2.out',
    stagger: 0.1
  });
});

document.querySelector('#reverseBtn2').addEventListener('click', () => {
  gsap.to(cards2, {
    opacity: 0, y: 50,
    duration: 0.5, ease: 'power2.out',
    stagger: 0.1
  });
});

How It Works

Use gsap.to() with the stagger property to efficiently animate multiple elements with smart timing delays.

Stagger Properties

  • stagger: 0.1 - Delay between each element's animation start
  • stagger: { each: 0.1 } - Same as above, explicit form
  • stagger: { from: 'end' } - Start from the last element

Performance

Stagger is optimized by GSAP to minimize layout thrashing. For 50+ elements, consider using will-change CSS property.
