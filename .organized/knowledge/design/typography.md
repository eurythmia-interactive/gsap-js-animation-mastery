# Typography

## Font Families (Used in Project)

| Font | Type | Use |
|------|------|-----|
| Playfair Display | Serif | Headlines, display |
| Bebas Neue | Sans | Bold headlines |
| Inter | Sans | Body text |
| Poppins | Sans | Modern UI |
| Cormorant Garamond | Serif | Creative, elegant |
| Abril Fatface | Serif | Dramatic display |

## Google Fonts Import

```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&family=Playfair+Display:wght@700&display=swap" rel="stylesheet">
```

## Typography Scale

| Class | Size |
|-------|------|
| text-xs | 0.75rem |
| text-sm | 0.875rem |
| text-base | 1rem |
| text-lg | 1.125rem |
| text-2xl | 1.5rem |
| text-4xl | 2.25rem |
| text-6xl | 3.75rem |

## Fluid Typography

```css
/* Clamp for responsive scaling */
font-size: clamp(1.5rem, 5vw, 3rem);
```
