JavaScript

JavaScript 
import { gsap } from 'gsap';
import { ScrollTrigger } from 'gsap/ScrollTrigger';

gsap.registerPlugin(ScrollTrigger);

// Animate each section item when it scrolls into view
gsap.from('.scroll-demo-item', {
  opacity: 0,
  y: 100,
  duration: 1,
  ease: "power2.out",
  stagger: 0.3,
  scrollTrigger: {
    trigger: '.scroll-demo-section',
    start: "top top",
    end: "bottom top",
    scrub: false,
    toggleActions: 'play none none reverse'
  }
});

How It Works

ScrollTrigger is GSAP's most powerful plugin. It links animations to scroll position, enabling parallax effects, scroll-driven storytelling, and reveal animations.

Key Concepts

trigger - The element that activates the animation
start/end - When to start/end relative to viewport
scrub - Link animation to scroll (true) or trigger once (false)

ScrollTrigger Options

Option Description Example
trigger Element that triggers animation '.section'
start When to start (element, viewport) 'top 80%'
end When to end 'top 50%'
scrub Link to scroll (true) or trigger once true or false
toggleActions What happens at start/end 'play none none reverse'

Toggle Actions

Four actions for four states: onEnter, onLeave, onEnterBack, onLeaveBack

  • play - Start animation
  • pause - Pause animation
  • reverse - Play backwards
  • reset - Jump to start
  • none - Do nothing

Common Patterns

  • Reveal on scroll: scrub: false, toggleActions: 'play none none reverse'
  • Parallax: scrub: true with y-axis movement
  • Progress bar: scrub: true with scaleX

Try It Yourself

Edit /src/lessons/lesson-15.js and experiment:

  • Show markers: Add markers: true to see trigger points
  • Change start position: Try 'top top' or 'center center'
  • Different stagger: Try stagger: 0.5 for slower cascade
  • Add rotation: Include rotation: -10 in the animation
