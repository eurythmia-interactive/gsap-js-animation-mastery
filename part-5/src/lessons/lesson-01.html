How It Works

Font pairing creates visual hierarchy through contrast. The key is combining fonts with distinct personalities:

Display Font (Playfair Display)

  • font-display → Playfair Display serif font
  • Use for headlines and hero text
  • Creates editorial, sophisticated feel
  • Bold weights (600-800) work best

Body Font (Inter)

  • font-sans → Inter sans-serif font
  • Use for paragraphs and UI elements
  • Highly readable at all sizes
  • Weights 400-600 for body, 700 for emphasis

Pairing Formula

  • Contrast Pair: Serif + Sans-Serif = Classic, editorial feel
  • Modern Pair: Geometric + Humanist = Clean, contemporary
  • Tech Pair: Sans-Serif + Monospace = Developer-friendly

Google Fonts Setup

Add to your HTML head:

<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=Playfair+Display:wght@600;700;800&display=swap" rel="stylesheet">
