How It Works

Form Best Practices

  • Minimize fields - Only ask for what you need
  • Clear labels - Placeholder text isn't enough
  • Visible focus states - Help users navigate
  • Immediate validation - Show errors as they type
  • Trust signals - Privacy notes, security badges

Input States

  • Default: border-zinc-700
  • Focus: focus:border-amber-500 focus:ring-1 focus:ring-amber-500
  • Error: border-red-500 focus:ring-red-500
  • Success: border-green-500 focus:ring-green-500
  • Disabled: opacity-50 cursor-not-allowed

Reducing Friction

  • Use email-only for simple lead capture
  • Progressive disclosure for longer forms
  • Show password requirements upfront
  • Offer "Show password" toggle
  • Auto-focus first field
