Lesson 07 Part 5: Landing Page Design

Call to Action

A call-to-action (CTA) is where conversion happens. Effective CTAs use contrast, clarity, and strategic placement to guide users toward the desired action. This lesson covers button hierarchy and optimization techniques.

Demo - Button Hierarchy

Primary CTA - Most prominent

Amber background, shadow glow, scale on hover

Secondary CTA - Less prominent

Outline style, neutral colors

Tertiary CTA - Minimal

Learn more about our features

Text link style, subtle presence

No credit card required • 14-day free trial

The Code

Button Hierarchy

HTML 
<!-- Primary CTA -->
<button class="bg-amber-500 hover:bg-amber-600 text-black font-bold px-10 py-4 rounded-full shadow-lg shadow-amber-500/25">
  Start Free Trial
</button>

<!-- Secondary CTA -->
<button class="border-2 border-zinc-600 hover:border-zinc-400 text-white font-semibold px-8 py-3 rounded-lg">
  View Demo
</button>

<!-- Tertiary CTA -->
<a class="text-amber-500 hover:text-amber-400 font-medium underline underline-offset-4">
  Learn more
</a>

<!-- Trust signals near CTA -->
<span class="flex items-center gap-1 text-sm text-zinc-500">
  <svg class="w-4 h-4 text-green-500" fill="currentColor" viewBox="0 0 20 20">...</svg>
  No credit card required
</span>

How It Works

CTA Design Principles

  • Contrast - Stand out from surrounding elements
  • Clarity - Action-oriented text ("Get Started", not "Submit")
  • Size - Large enough to be noticed
  • Space - Surrounding whitespace helps focus
  • Feedback - Hover/focus states provide feedback

CTA Text Best Practices

  • Use first-person: "Get my free trial" not "Get your free trial"
  • Be specific: "Start free trial" not "Click here"
  • Create urgency: "Start now" or "Get access"
  • Focus on value: "Get started free" not "Sign up"

Trust Signals to Pair with CTAs

  • "No credit card required"
  • "14-day free trial"
  • "Cancel anytime"
  • Security badges/lock icons
  • Money-back guarantee
