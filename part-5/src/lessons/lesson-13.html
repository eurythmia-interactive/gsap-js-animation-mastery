White Space Usage

White space (or negative space) is the empty area around elements. It's not "wasted" space—it's a powerful design tool that creates breathing room, guides attention, and communicates elegance and quality.

The Code

Generous White Space

HTML 
<!-- Generous padding on container -->
<div class="bg-zinc-800 rounded-xl p-8">
  
  <!-- Cards with gap -->;
  <div class="flex flex-wrap justify-center gap-4">
    <div class="bg-zinc-700 px-6 py-4 rounded-lg text-center">
      Feature A
    </div>
  </div>
</div>

<!-- Minimal section with max-width -->
<div class="text-center py-16">
  <h3 class="text-2xl font-display text-white mb-4">
    Less Elements
  </h3>
  <p class="text-zinc-500 max-w-md mx-auto mb-8">
    Description
  </p>
</div>

How It Works

White Space Benefits

  • Focus - Draws attention to what matters
  • Clarity - Makes content easier to scan
  • Elegance - Creates premium feel
  • Breathing room - Reduces cognitive load
  • Hierarchy - Naturally establishes importance

White Space Techniques

  • Increase section padding (py-16, py-24)
  • Use max-w-* to constrain text width
  • Add gap-* between elements
  • Use margins to separate content blocks
  • Center content with mx-auto

Common Mistakes

  • Filling every pixel with content
  • Cramming too many elements together
  • Ignoring mobile spacing needs
  • Using same spacing everywhere
  • Thinking white space is wasted space
