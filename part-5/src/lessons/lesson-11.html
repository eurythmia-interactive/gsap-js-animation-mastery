Lesson 11 Part 5: Landing Page Design

Vertical Rhythm

Vertical rhythm creates visual harmony through consistent spacing between elements. Just as musicians keep a steady beat, designers use consistent spacing to create a sense of order and professionalism.

Demo - Consistent Spacing Scale

TIGHT (py-8 = 32px)

Related content grouping

Items that belong together with minimal breathing room

STANDARD (py-16 = 64px)

Default section spacing

Most content should use this spacing level

RELAXED (py-20 = 80px)

Major sections need more room

Important transitions deserve extra breathing space

SPACIOUS (py-24 = 96px)

Hero to content transitions

Maximum drama and focus between major sections

The Code

Spacing Scale

HTML 
<!-- Tight - Related content (32px) -->
<div class="py-8">
  <h3 class="text-xl mb-3">Related content</h3>
</div>

<!-- Standard - Default sections (64px) -->
<div class="py-16">
  <h3 class="text-2xl mb-4">Section title</h3>
</div>

<!-- Relaxed - Major sections (80px) -->
<div class="py-20">
  <h2 class="text-3xl mb-6">Important section</h2>
</div>

<!-- Spacious - Transitions (96px) -->
<div class="py-24">
  <h2 class="text-4xl mb-8">Hero transition</h2>
</div>

How It Works

Spacing Scale Reference

  • py-8 → 2rem (32px) - Tight, related items
  • py-12 → 3rem (48px) - Compact, cards
  • py-16 → 4rem (64px) - Standard sections
  • py-20 → 5rem (80px) - Relaxed, major sections
  • py-24 → 6rem (96px) - Spacious, transitions
  • py-32 → 8rem (128px) - Dramatic, hero

Vertical Rhythm Benefits

  • Creates professional, polished appearance
  • Helps users predict where content will be
  • Makes scanning easier
  • Establishes visual hierarchy naturally

Best Practices

  • Pick 3-4 spacing values and stick to them
  • Hero sections need most space
  • Related items grouped tightly
  • Section transitions need breathing room
  • Use consistent padding within cards
