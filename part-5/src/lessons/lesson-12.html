Grid Layouts

HTML 
<!-- 2 Column -->
<div class="grid grid-cols-1 md:grid-cols-2 gap-6">
  <div>Column 1</div>
  <div>Column 2</div>
</div>

<!-- 3 Column -->
<div class="grid grid-cols-1 md:grid-cols-3 gap-6">
  <div>Feature 1</div>
  <div>Feature 2</div>
  <div>Feature 3</div>
</div>

<!-- Asymmetric 2:1 -->
<div class="grid grid-cols-1 lg:grid-cols-3 gap-6">
  <div class="lg:col-span-2">Main Content</div>
  <div>Sidebar</div>
</div>

<!-- Feature + Visual (vertically centered) -->
<div class="grid grid-cols-1 lg:grid-cols-2 gap-12 items-center">
  <div>Feature</div>
  <div>Visual</div>
</div>

How It Works

Grid Column Classes

  • grid-cols-1 → 1 column (mobile default)
  • grid-cols-2 → 2 equal columns
  • grid-cols-3 → 3 equal columns
  • grid-cols-4 → 4 equal columns
  • grid-cols-12 → 12-column grid

Column Spanning

  • col-span-2 → Span 2 columns
  • col-span-3 → Span 3 columns
  • col-span-full → Full width
  • lg:col-span-2 → Responsive span

Content Width Guidelines

  • max-w-2xl (640px) - Narrow, focused text
  • max-w-3xl (768px) - Blog posts, value props
  • max-w-5xl (1024px) - Most content
  • max-w-6xl (1280px) - Dashboards, wide layouts
  • max-w-7xl (1536px) - Maximum content area
