Optimal: 45-65 Characters

Reading Guidelines
• Line length: 45-65 characters (max-w-[65ch])
• Line height: 1.5-1.7 for body text
• Letter spacing: Tighter for headlines
• Justify: Avoid on mobile
Key Code
<!-- Optimal reading width -->
<article class="max-w-[65ch] mx-auto p-4">
  
  <!-- Headline - tight tracking -->
  <h1 class="text-2xl font-bold 
               tracking-tight text-stone-900">
    Article Title
  </h1>
  
  <!-- Body text - relaxed leading -->
  <div class="text-stone-700 
               leading-[1.7] space-y-4">
    <p>
      Body text optimized for comfortable reading.
    </p>
  </div>
</article>
Optimal Values
/* Line length (characters per line) */
max-width: 65ch;  /* Optimal: 45-65 */
max-width: 75ch;  /* Comfortable: 60-75 */
max-width: 40ch;  /* Narrow: 30-40 */

/* Line height */
line-height: 1.4;  /* Tight - headlines */
line-height: 1.5;  /* Normal - UI */
line-height: 1.6;  /* Relaxed - body */
line-height: 1.7;  /* Extra relaxed */

/* Letter spacing */
tracking-tight;   /* -0.025em - headlines */
tracking-normal;   /* 0 - body */
tracking-wide;     /* 0.025em - captions */
