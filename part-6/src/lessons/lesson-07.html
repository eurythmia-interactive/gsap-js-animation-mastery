clamp() Formula
clamp(min, preferred, max)
Min 1.75rem
Preferred 5vw + 1rem
Max 2.5rem
Key Code
<style>
  /* Fluid headline */
  .fluid-headline {
    font-size: clamp(1.75rem, 5vw + 1rem, 2.5rem);
    line-height: 1.1;
  }

  /* Fluid subheadline */
  .fluid-subhead {
    font-size: clamp(1.125rem, 2.5vw + 0.5rem, 1.5rem);
    line-height: 1.3;
  }

  /* Fluid body */
  .fluid-body {
    font-size: clamp(0.875rem, 1.5vw + 0.5rem, 1.125rem);
    line-height: 1.6;
  }
</style>

<!-- Usage -->
<h1 class="fluid-headline font-bold">
  Responsive Headline
</h1>
