Key Principles
/* 1. True black saves battery on OLED */
background-color: #000000;

/* 2. Elevation through lightness (not shadows) */
/* In dark mode, lighter surfaces = higher */

/* 3. Avoid pure white for body text */
/* Use zinc-100 or zinc-200 instead */
color: #fafafa;  /* zinc-100 */

/* 4. Use subtle borders to separate */
border: 1px solid #27272a;  /* zinc-800 */
