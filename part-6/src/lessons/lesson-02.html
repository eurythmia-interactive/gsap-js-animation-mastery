Guidelines
/* Apple HIG */
min-size: 44pt × 44pt (≈44×44px at 1x)

/* WCAG 2.1 Level AA */
min-size: 44px × 44px

/* Material Design */
min-size: 48dp (48 CSS pixels)

/* Gap between targets */
gap: 8px - 16px
