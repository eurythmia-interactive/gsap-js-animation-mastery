The Code

Borders

HTML 
<!-- Border width: border, border-2, border-4 -->
<div class="border-2">2px border</div>

<!-- Border radius: rounded-sm to rounded-full -->
<div class="rounded-lg">Rounded</div>
<div class="rounded-full">Circle/Pill</div>

<!-- Border color -->
<div class="border-2 border-emerald-500">Emerald border</div>

Shadows

HTML 
<!-- Shadow sizes: shadow-sm, shadow, shadow-md, shadow-lg, shadow-xl, shadow-2xl -->
<div class="shadow-lg">Large shadow</div>

<!-- Custom shadow -->
<div class="shadow-[0_4px_20px_rgba(16,185,129,0.4)]">Glowing shadow</div>

<!-- No shadow -->
<div class="shadow-none">No shadow</div>

Quick Reference

Border Radius

  • rounded-none - 0
  • rounded-sm - 0.125rem
  • rounded - 0.25rem (default)
  • rounded-md - 0.375rem
  • rounded-lg - 0.5rem
  • rounded-xl - 0.75rem
  • rounded-2xl - 1rem
  • rounded-full - 9999px (circle)

Common Patterns

  • border border-gray-300 rounded-lg - Card with subtle border
  • rounded-full w-12 h-12 - Circular icon button
  • shadow-lg rounded-xl - Elevated card
  • hover:shadow-xl transition-shadow - Hover lift effect
