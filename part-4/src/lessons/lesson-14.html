The Code

Cursor Styles

HTML 
<!-- Cursor variants -->
<button class="cursor-pointer">Pointer</button>
<div class="cursor-default">Default</div>
<div class="cursor-move">Move</div>
<div class="cursor-grab">Grab</div>
<button class="cursor-not-allowed">Not Allowed</button>

Pointer Events & Selection

HTML 
<!-- Pointer events (click handling) -->
<div class="pointer-events-none">Cannot be clicked</div>
<div class="pointer-events-auto">Can be clicked (default)</div>

<!-- User selection -->
<p class="select-none">Cannot select</p>
<p class="select-text">Can select (default)</p>
<p class="select-all">Click to select all</p>

Quick Reference

Cursor Styles

  • cursor-pointer - Hand (links, buttons)
  • cursor-default - Arrow (default)
  • cursor-move - Move indicator
  • cursor-grab - Grab (for draggable)
  • cursor-not-allowed - Not allowed
  • cursor-wait - Loading spinner

Common Use Cases

  • select-none - Game elements, icons
  • select-all - Code blocks, terminal
  • pointer-events-none - Overlay elements
  • cursor-grab - Draggable items

Overflow Values

  • overflow-auto - Scroll when needed
  • overflow-hidden - No scroll, clipped
  • overflow-scroll - Always show scrollbars
  • overflow-visible - Show outside container
