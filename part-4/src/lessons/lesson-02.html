The Code

Spacing Classes

HTML 
<!-- Margin: m-4 (1rem = 16px) -->
<div class="m-4 bg-emerald-500">Margin</div>

<!-- Padding: p-6 (1.5rem = 24px) -->
<div class="p-6 bg-blue-500">Padding</div>

<!-- Horizontal margin auto (centering) -->
<div class="mx-auto w-1/2">Centered</div>

Sizing Classes

HTML 
<!-- Width: w-1/2 (50%), w-full (100%), w-screen (100vw) -->
<div class="w-1/2">50% Width</div>

<!-- Height: h-24 (6rem = 96px) -->
<div class="h-24">Tall</div>

<!-- Min height: min-h-screen (100vh) -->
<div class="min-h-screen">Full Height</div>

Spacing Scale

Tailwind uses a consistent 4px-based scale. Each number represents the multiplier:

Quick Reference

  • 1 = 0.25rem (4px)
  • 2 = 0.5rem (8px)
  • 4 = 1rem (16px)
  • 6 = 1.5rem (24px)
  • 8 = 2rem (32px)
  • 12 = 3rem (48px)
  • 16 = 4rem (64px)

Directional Modifiers

  • mt-4 - margin-top
  • mb-4 - margin-bottom
  • ml-4 - margin-left
  • mr-4 - margin-right
  • mx-4 - margin-left + margin-right
  • my-4 - margin-top + margin-bottom

Width Fractions

  • w-1/2 = 50%
  • w-1/3 = 33.333%
  • w-2/3 = 66.666%
  • w-1/4 = 25%
  • w-3/4 = 75%
