The Code

Position Properties

HTML 
<!-- Position values -->
<div class="relative">Relative to itself</div>
<div class="absolute">Relative to parent</div>
<div class="fixed">Fixed to viewport</div>
<div class="sticky">Sticky until scrolled past</div>

Placement

HTML 
<!-- Inset (top/right/bottom/left) -->
<div class="absolute inset-0">Full coverage</div>
<div class="absolute top-0 left-0">Top left</div>
<div class="absolute bottom-4 right-4">Bottom right with offset</div>

<!-- Z-index -->
<div class="relative z-10">Layer 10</div>
<div class="relative -z-10">Behind (negative)</div>

Quick Reference

Position Values

  • static - Default (normal flow)
  • relative - Positioned relative to normal
  • absolute - Removed from flow, relative to positioned parent
  • fixed - Fixed to viewport window
  • sticky - Sticks when scrolling past

Centering Patterns

Transform translate:

absolute left-1/2 top-1/2 -translate-x-1/2 -translate-y-1/2

Flexbox:

absolute inset-0 flex items-center justify-center

Z-Index Scale

z-0, z-10, z-20, z-30, z-40, z-50, z-auto

Use -z-10 for negative values to push behind.
