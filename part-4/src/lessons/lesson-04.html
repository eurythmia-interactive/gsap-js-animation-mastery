The Code

Font Size & Weight

HTML 
<!-- Font sizes: text-xs through text-9xl -->
<h1 class="text-4xl">Large Heading</h1>
<p class="text-sm text-gray-400">Small muted text</p>

<!-- Font weights: font-thin (100) to font-black (900) -->
<p class="font-bold">Bold text</p>
<p class="font-semibold">Semibold text</p>

Line Height & Letter Spacing

HTML 
<!-- Line height: leading-none to leading-loose -->
<p class="leading-relaxed">Relaxed line height</p>

<!-- Letter spacing: tracking-tighter to tracking-widest -->
<p class="tracking-widest uppercase">SPACED OUT</p>

<!-- Text alignment -->
<p class="text-center">Centered text</p>

Typography Summary

Font Sizes

  • text-xs = 0.75rem (12px)
  • text-sm = 0.875rem (14px)
  • text-base = 1rem (16px) - default
  • text-lg = 1.125rem (18px)
  • text-xl = 1.25rem (20px)
  • text-2xl to text-9xl

Font Families

  • font-sans - Sans-serif (default)
  • font-serif - Serif
  • font-mono - Monospace

Common Combinations

  • text-sm text-gray-400 - Muted secondary text
  • text-lg font-semibold - Subheadings
  • text-3xl font-bold tracking-tight - Hero titles
