Feature Card HTML

HTML 
<div class="bg-gray-800 rounded-xl overflow-hidden border border-gray-700 hover:border-emerald-500/50 transition-all duration-300 hover:shadow-xl">
  
  <!-- Image/Icon Area -->
  <div class="h-48 bg-gradient-to-br from-emerald-400 to-teal-600 flex items-center justify-center">
    <svg class="... icon ..."></svg>
  </div>
  
  <!-- Content -->
  <div class="p-6">
    <h3 class="text-xl font-bold text-white mb-2">Title</h3>
    <p class="text-gray-400 mb-4">Description text</p>
    <button class="w-full bg-emerald-500 hover:bg-emerald-600 text-white font-semibold py-2 px-4 rounded-lg transition-colors duration-200">
      Button Text
    </button>
  </div>
</div>

Key Concepts Used

Layout

  • flex, flex-col, flex-wrap - Container layouts
  • items-center, justify-center - Alignment
  • gap-2, gap-4 - Spacing
  • w-full, flex-1 - Sizing

Styling

  • bg-gray-800 - Dark card background
  • rounded-xl - Large border radius
  • border, border-gray-700 - Subtle border
  • shadow-lg - Elevation

Interactivity

  • hover:border-emerald-500/50 - Border color change
  • hover:shadow-xl - Shadow enhancement
  • transition-all duration-300 - Smooth transitions
  • group + group-hover: - Grouped hover effects
