Lesson 09 Part 4: Tailwind CSS Basics

Custom Values

Use Tailwind's arbitrary syntax [] for custom one-off values that aren't in the default scale.

Demo - Arbitrary Values

p-[22px] (custom pixel value)

Custom 22px padding

w-[200px] (custom width)

200px wide

bg-[#ff6b35] (custom hex color)

Custom orange color

Complex arbitrary values

calc() width + custom padding

hover:bg-[#ff8c55] (hover with custom color)

The Code

Arbitrary Values

HTML 
<!-- Custom padding: p-[22px] -->
<div class="p-[22px]">Custom padding</div>

<!-- Custom width: w-[200px] -->
<div class="w-[200px]">200px wide</div>

<!-- Custom color: bg-[#ff6b35] -->
<div class="bg-[#ff6b35]">Custom orange</div>

<!-- Custom calc: w-[calc(100%-2rem)] -->
<div class="w-[calc(100%-2rem)]">Calc width</div>

When to Use Arbitrary Values

Good Use Cases

  • One-off values not in Tailwind's scale
  • Brand-specific colors (e.g., bg-[#your-brand])
  • Pixel-perfect designs with specific measurements
  • Complex calculations

Best Practice

Don't overuse arbitrary values! If you find yourself using the same arbitrary value multiple times, consider adding it to your Tailwind config instead:

tailwind.config = { theme: { extend: { colors: { brand: '#ff6b35' }}}}

Arbitrary Properties

For CSS properties without Tailwind utilities:

class="[aspect-ratio:16/9]"

This generates: aspect-ratio: 16/9
