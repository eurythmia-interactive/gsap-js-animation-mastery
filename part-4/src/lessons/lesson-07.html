The Code

Responsive Prefixes

HTML 
<!-- Mobile first: base styles apply to all, then override at breakpoints -->
<div class="text-center md:text-left">
  Centered on mobile, left on tablet+
</div>

<!-- Responsive grid: 1 col mobile, 2 tablet, 4 desktop -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-4">

Showing/Hiding Elements

HTML 
<!-- Hidden on mobile, shown from md (768px+) -->
<div class="hidden md:block">Visible on tablet+</div>

<!-- Shown on mobile, hidden from md -->
<div class="block md:hidden">Visible only on mobile</div>

Breakpoints

Tailwind Breakpoints

  • sm: - 640px (Large phones)
  • md: - 768px (Tablets)
  • lg: - 1024px (Laptops)
  • xl: - 1280px (Desktops)
  • 2xl: - 1536px (Large screens)

Mobile First Philosophy

Styles without a prefix apply to all screen sizes. Then you add prefixes to progressively enhance for larger screens.

flex-col md:flex-row = Stack on mobile, row on tablet+

Common Patterns

  • grid-cols-1 md:grid-cols-2 lg:grid-cols-3 - Responsive card grid
  • flex-col md:flex-row - Responsive flex layout
  • hidden md:block - Hide on mobile, show on tablet+
  • w-full md:w-1/2 - Full width mobile, half desktop
