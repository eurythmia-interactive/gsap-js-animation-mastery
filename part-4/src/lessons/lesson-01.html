Lesson 01 Part 4: Tailwind CSS Basics

What is Tailwind?

Tailwind CSS is a utility-first CSS framework that provides low-level utility classes to build custom designs directly in your HTML.

Demo - Traditional CSS vs Tailwind

Traditional CSS

.my-button {

background-color: #10b981;

color: white;

padding: 1rem 2rem;

border-radius: 0.5rem;

font-weight: 600;

}

Tailwind CSS

The Code

HTML with Tailwind Classes

HTML 
<button class="bg-emerald-500 hover:bg-emerald-600 text-white px-6 py-3 rounded-lg font-semibold transition-colors duration-200">
  Click me - Hover me!
</button>

How It Works

Each class in Tailwind is a single CSS property. You compose them directly in your HTML:

Utility Classes Explained

  • bg-emerald-500 → background-color: #10b981
  • hover:bg-emerald-600 → darker green on hover
  • text-white → color: white
  • px-6 → padding-left & padding-right: 1.5rem
  • py-3 → padding-top & padding-bottom: 0.75rem
  • rounded-lg → border-radius: 0.5rem
  • font-semibold → font-weight: 600
  • transition-colors → animate color changes

Play CDN Setup

Add this script to your HTML to use Tailwind without any build step:

<script src="https://cdn.tailwindcss.com"></script>

Why Tailwind?

  • No naming struggles - no need for .button-primary
  • Consistent spacing, colors, and shadows
  • Responsive design built-in
  • Small CSS file size (tree-shaking in production)
