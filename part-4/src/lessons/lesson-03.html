The Code

Text & Background Colors

HTML 
<!-- Text colors -->
<p class="text-white">White text</p>
<p class="text-emerald-500">Emerald text</p>

<!-- Background colors -->
<div class="bg-gray-800">Dark background</div>
<div class="bg-blue-500">Blue background</div>

Opacity Variants

HTML 
<!-- Using opacity-* class -->
<div class="bg-black opacity-50">50% opacity</div>

<!-- Using slash notation (modern) -->
<div class="bg-black/50">50% opacity</div>
<p class="text-white/75">75% text opacity</p>

Color Shades

Most colors in Tailwind have shades from 50 (lightest) to 900 (darkest):

Gray Scale Example

  • text-gray-50 - Lightest (near white)
  • text-gray-100, text-gray-200, text-gray-300
  • text-gray-400 - Default body text gray
  • text-gray-500 - Medium gray
  • text-gray-600, text-gray-700, text-gray-800
  • text-gray-900 - Darkest (near black)

Slash Notation

The slash notation bg-emerald-500/50 is modern and concise. It sets the opacity at the same time as the color, making it easier to read.

Common Colors

Tailwind includes: gray, red, orange, yellow, green, emerald, teal, blue, sky, cyan, purple, pink, and more!
