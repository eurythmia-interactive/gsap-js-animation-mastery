The Code

Dark Mode Setup

HTML 
<!-- Enable class-based dark mode -->
<script>
  tailwind.config = {
    darkMode: 'class'
  }
</script>

Using dark: Prefix

HTML 
<!-- Background: white on light, gray-800 on dark -->
<div class="bg-white dark:bg-gray-800">

<!-- Text: gray-900 on light, gray-100 on dark -->
<p class="text-gray-900 dark:text-gray-100">Adaptive text</p>

<!-- Colors can shift slightly between modes -->
<button class="bg-emerald-500 dark:bg-emerald-600">Button</button>

Dark Mode Best Practices

Enabling Dark Mode

  • Class-based (recommended): darkMode: 'class'
  • Add dark class to <html> to activate
  • Toggle via JavaScript: document.documentElement.classList.toggle('dark')

Color Guidelines

  • Don't use pure black (#000) - use gray-900 or gray-800
  • Shift colors slightly: blue-500 → blue-600 for dark mode
  • Reduce shadow intensity in dark mode
  • Use slightly muted backgrounds for better contrast

Manual Toggle Example

document.documentElement.classList.toggle('dark')

Toggle this on a button click to switch modes.
