The Code

Basic Grid

HTML 
<!-- 3 column grid with gaps -->
<div class="grid grid-cols-3 gap-4">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

Spanning Columns

HTML 
<!-- 4 column grid with spanning -->
<div class="grid grid-cols-4 gap-4">
  <div class="col-span-2">Spans 2 columns</div>
  <div>Auto</div>
  <div>Auto</div>
</div>

Grid vs Flexbox

When to Use Grid

  • Two-dimensional layouts (rows AND columns)
  • Card layouts, galleries, dashboards
  • When items need to line up both horizontally and vertically
  • Known number of columns

When to Use Flexbox

  • One-dimensional layouts (row OR column)
  • Navigation bars, button groups
  • Content that wraps naturally
  • Unknown number of items

Column Classes

  • grid-cols-1 to grid-cols-12
  • col-span-2 - Span 2 columns
  • col-span-full - Span entire row
  • gap-4 - 1rem gap between items
