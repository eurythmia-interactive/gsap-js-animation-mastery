The Code

State Variants

HTML 
<!-- Hover: changes on mouse over -->
<button class="bg-emerald-500 hover:bg-emerald-600">
  Hover Me
</button>

<!-- Focus: when element has focus -->
<input class="focus:ring-2 focus:ring-emerald-500 focus:outline-none">

<!-- Active: while being clicked -->
<button class="active:bg-blue-700">
  Press Me
</button>

<!-- Group hover: child changes when parent is hovered -->
<div class="group p-4 hover:bg-gray-700">
  <p class="group-hover:text-emerald-400">Text</p>
</div>

State Variants Reference

Common State Variants

  • hover: - Mouse over element
  • focus: - Element has focus
  • active: - Element is being clicked
  • disabled: - Element is disabled
  • group-hover: - When parent group is hovered
  • focus-within: - When any child has focus

Combining States

You can combine multiple state variants:

bg-blue-500 hover:bg-blue-700 focus:ring-2 active:bg-blue-800

Focus Ring

For accessibility, always include focus states:

focus:outline-none focus:ring-2 focus:ring-emerald-500
