The Code

Transition Properties

HTML 
<!-- Transition property variants -->
<div class="transition">All properties (default)</div>
<div class="transition-colors">Color transitions only</div>
<div class="transition-opacity">Opacity transitions only</div>
<div class="transition-transform">Transform transitions only</div>
<div class="transition-all">All properties including size</div>

Duration & Easing

HTML 
<!-- Duration: duration-75 to duration-2000 -->
<div class="duration-300">300ms (default)</div>

<!-- Easing: ease-linear, ease-in, ease-out, ease-in-out -->
<div class="ease-out">Slow end (default)</div>

<!-- Combined -->
<button class="transition-all duration-300 ease-in-out">Smooth</button>

<!-- Common hover pattern -->
<button class="transition-transform duration-200 hover:scale-105">Scale on hover</button>

Quick Reference

Transition Properties

  • transition - All properties (shorthand)
  • transition-colors - background, border, color, text
  • transition-opacity - opacity
  • transition-transform - transform
  • transition-all - all properties + dimensions

Duration

Values: duration-75, duration-100, duration-150, duration-200, duration-300, duration-500, duration-700, duration-1000

Common Patterns

  • transition-colors duration-200 hover:bg-blue-600 - Button hover
  • transition-transform duration-300 hover:scale-105 - Card lift
  • transition-all duration-300 hover:shadow-xl - Shadow lift
