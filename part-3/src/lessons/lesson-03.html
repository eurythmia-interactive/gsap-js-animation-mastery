Word Stagger

Learn how to split text into individual words and animate them with elastic bounce effects for playful, attention-grabbing reveals.

JavaScript

JavaScript 
import { gsap } from 'gsap';
import { splitText, unsplitText } from '../../../utils/splitText.js';

const title = document.querySelector('#title');
const playBtn = document.querySelector('#playBtn');
const resetBtn = document.querySelector('#resetBtn');

// Split title into words on page load
const words = splitText(title, 'words');

// Add color classes to each word for playful rainbow effect
words.forEach((word, index) => {
  word.classList.add(`word-${index % 3}`);
});

// Set initial hidden state
gsap.set(words, {
  opacity: 0,
  y: 100,
  scale: 0,
  rotationX: -90
});

// Play button - animate words in with elastic bounce
playBtn.addEventListener('click', () => {
  gsap.fromTo(words,
    {
      opacity: 0,
      y: 100,
      scale: 0,
      rotationX: -90
    },
    {
      opacity: 1,
      y: 0,
      scale: 1,
      rotationX: 0,
      stagger: 0.15,
      ease: 'elastic.out(1, 0.5)'
    }
  );
});

// Reset button - return to initial hidden state
resetBtn.addEventListener('click', () => {
  gsap.to(words, {
    opacity: 0,
    y: 100,
    scale: 0,
    rotationX: -90,
    stagger: 0.08,
    ease: 'power2.in'
  });
});

How It Works

Word staggering creates dramatic reveals by animating each word sequentially with elastic bounce. The rotationX property adds a 3D flip effect that enhances the bounce.

Word Splitting

  • splitText(element, 'words') - Splits text into word spans
  • Each word becomes its own animatable element
  • Preserves spacing between words

Elastic Easing

The elastic.out(1, 0.5) easing creates a bouncy, playful effect. The parameters control the amplitude (1) and stiffness (0.5). Higher amplitude = more bounce, lower stiffness = longer bounce duration.

Stagger Timing

A stagger of 0.15 means each word starts 150ms after the previous one. Combined with elastic easing, this creates a wave-like cascade of bounces.

3D Rotation Effect

Using rotationX creates a flip effect. Remember that 3D transforms require proper perspective. Add perspective: 1000 to the parent container for best results.
