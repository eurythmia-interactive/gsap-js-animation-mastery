Lesson 05 Part 3: SVG Text Animations

SVG Draw-On Effect

Learn how to animate SVG strokes drawing on using strokeDasharray and strokeDashoffset.

Demo - SVG Draw-On

The Code

SVG

SVG 
<svg viewBox="0 0 600 150" xmlns="http://www.w3.org/2000/svg">
  <defs>
    <linearGradient id="cyanPurpleGradient" x1="0%" y1="0%" x2="100%" y2="0%">
      <stop offset="0%" stop-color="#00d4ff" />
      <stop offset="100%" stop-color="#8b5cf6" />
    </linearGradient>
  </defs>
  <path id="drawPath" d="M50,75 Q150,25 250,75 T450,75 T650,75"
        class="draw-path gradient-stroke glow-stroke" />
</svg>

JavaScript

JavaScript 
import { gsap } from 'gsap';

const drawPath = document.querySelector('#drawPath');
const playBtn = document.querySelector('#playBtn');
const resetBtn = document.querySelector('#resetBtn');

// Get the total length of the path
const length = drawPath.getTotalLength();

// Set initial state: stroke is invisible
gsap.set(drawPath, {
  strokeDasharray: length,
  strokeDashoffset: length
});

// Play button - animate stroke drawing on
playBtn.addEventListener('click', () => {
  gsap.fromTo(drawPath,
    { strokeDashoffset: length },
    {
      strokeDashoffset: 0,
      duration: 2,
      ease: 'power2.inOut'
    }
  );
});

// Reset button - return stroke to hidden state
resetBtn.addEventListener('click', () => {
  gsap.to(drawPath, {
    strokeDashoffset: length,
    duration: 0.5,
    ease: 'power2.in'
  });
});

How It Works

SVG strokes can be animated to create a "drawing" effect using two key properties: strokeDasharray and strokeDashoffset.

strokeDasharray

Defines the pattern of dashes and gaps. Setting it equal to the path length means the entire stroke is one continuous dash.

strokeDashoffset

The distance into the dash pattern to start. By animating from full length to 0, the stroke appears to "draw on".

getTotalLength()

This SVG DOM method returns the total length of the path, allowing us to set up the dash pattern precisely.

Works on Paths Only

getTotalLength() works on <path>, <rect>, <circle>, etc., but NOT on <text> elements.
