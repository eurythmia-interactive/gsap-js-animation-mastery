Lesson 05 Part 3: SVG Text Animations

SVG Draw-On Effect

Learn how to animate SVG text strokes drawing on using strokeDasharray and strokeDashoffset.

Demo - SVG Draw-On

The Code

HTML

HTML 
<svg viewBox="0 0 600 120" xmlns="http://www.w3.org/2000/svg">
  <defs>
    <linearGradient id="cyanPurpleGradient" x1="0%" y1="0%" x2="100%" y2="0%">
      <stop offset="0%" stop-color="#00d4ff" />
      <stop offset="100%" stop-color="#8b5cf6" />
    </linearGradient>
  </defs>
  <text id="drawText" x="50%" y="75" text-anchor="middle" class="draw-text gradient-stroke">
    DRAW ON
  </text>
</svg>

JavaScript

JavaScript 
import { gsap } from 'gsap';
import { splitText, unsplitText } from '../../../utils/splitText.js';

const drawText = document.querySelector('#drawText');
const playBtn = document.querySelector('#playBtn');
const resetBtn = document.querySelector('#resetBtn');

// Get the total length of the text path/stroke
const length = drawText.getTotalLength();

// Set initial state: stroke is invisible (dashoffset = length)
gsap.set(drawText, {
  strokeDasharray: length,
  strokeDashoffset: length
});

// Play button - animate stroke drawing on
playBtn.addEventListener('click', () => {
  gsap.fromTo(drawText,
    {
      strokeDashoffset: length
    },
    {
      strokeDashoffset: 0,
      duration: 3,
      ease: 'power2.inOut'
    }
  );
});

// Reset button - return stroke to hidden state
resetBtn.addEventListener('click', () => {
  gsap.to(drawText, {
    strokeDashoffset: length,
    duration: 0.5,
    ease: 'power2.in'
  });
});

How It Works

SVG strokes can be animated to create a "drawing" effect using two key properties: strokeDasharray and strokeDashoffset.

strokeDasharray

Defines the pattern of dashes and gaps used to paint the outline of the shape. Setting it equal to the path length means the entire stroke is made of one continuous dash.

strokeDashoffset

The distance into the dash pattern to start the dash. By animating this from the full length to 0, the stroke appears to "draw on".

getTotalLength()

This SVG DOM method returns the total length of the path or text outline, allowing us to precisely set up the dash pattern.

SVG Text Requirements

For the draw-on effect to work, the SVG text must have fill: transparent and a visible stroke. The fill hides the text interior while the stroke traces the outline.

Gradient Strokes

Apply stroke: url(#gradientId) on the SVG element to create gradient strokes that transition between colors as the line draws on.
