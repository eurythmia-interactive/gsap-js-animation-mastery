Lesson 05 Part 3: SVG Text Animations

SVG Draw-On Effect

Learn how to animate SVG text strokes drawing on using strokeDasharray and strokeDashoffset.

Demo - SVG Draw-On

The Code

SVG

SVG 
<svg viewBox="0 0 600 150" xmlns="http://www.w3.org/2000/svg">
  <defs>
    <linearGradient id="textGradient" x1="0%" y1="0%" x2="100%" y2="0%">
      <stop offset="0%" stop-color="#00d4ff" />
      <stop offset="100%" stop-color="#8b5cf6" />
    </linearGradient>
  </defs>
  
  <!-- Each letter is a <path> element -->
  <path class="draw-path" d="M30,30 L30,120 ..." />
  <path class="draw-path" d="M130,30 ..." />
  <!-- ... more letters ... -->
</svg>

JavaScript

JavaScript 
import { gsap } from 'gsap';

const drawPaths = document.querySelectorAll('.draw-path');
const playBtn = document.querySelector('#playBtn');
const resetBtn = document.querySelector('#resetBtn');

// Get lengths for each path
const lengths = Array.from(drawPaths).map(path => path.getTotalLength());

// Set initial state: all strokes invisible
drawPaths.forEach((path, i) => {
  gsap.set(path, {
    strokeDasharray: lengths[i],
    strokeDashoffset: lengths[i]
  });
});

// Play: animate strokes drawing on
playBtn.addEventListener('click', () => {
  gsap.fromTo(drawPaths,
    { strokeDashoffset: (i) => lengths[i] },
    {
      strokeDashoffset: 0,
      duration: 2.5,
      stagger: 0.1,
      ease: 'power2.inOut'
    }
  );
});

// Reset: hide strokes again
resetBtn.addEventListener('click', () => {
  gsap.to(drawPaths, {
    strokeDashoffset: (i) => lengths[i],
    duration: 0.5,
    stagger: 0.05,
    ease: 'power2.in'
  });
});

How It Works

SVG strokes can be animated to create a "drawing" effect. We use pre-converted path data for each letter so we can measure the total length with getTotalLength().

Converting Text to Paths

In a design tool like Figma or Illustrator:

  • Type your text
  • Select Outline Stroke (Figma) or Create Outlines (Illustrator)
  • Copy the SVG path data

strokeDasharray & strokeDashoffset

Setting both to the path's total length means the entire stroke is one dash. Animating the offset from length to 0 makes it "draw on".

Why Not <text> Element?

SVG's <text> element doesn't support getTotalLength(). That's why we convert to paths!
