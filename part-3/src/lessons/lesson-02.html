Lesson 02 Part 3: Text Animations

Typewriter Terminal

Learn how to create classic terminal-style typing effects using GSAP's free TextPlugin with a blinking cursor animation.

JavaScript

JavaScript 
import { gsap } from 'gsap';
import { TextPlugin } from 'gsap/TextPlugin';

// Register the free TextPlugin
gsap.registerPlugin(TextPlugin);

const terminalText = document.querySelector('#terminal-text');
const cursor = document.querySelector('#cursor');
const playBtn = document.querySelector('#playBtn');
const resetBtn = document.querySelector('#resetBtn');

const terminalMessage = `Welcome to GSAP Terminal

Type anything and watch it
appear letter by letter.

> System ready.`;

// Set initial state - hide cursor during typing
let typingTimeline;

// Play button - type out text with cursor
playBtn.addEventListener('click', () => {
  // Kill any existing animation
  if (typingTimeline) typingTimeline.kill();
  
  // Clear any existing text
  terminalText.textContent = '';
  
  // Create typing animation timeline
  typingTimeline = gsap.timeline();
  
  // Type text with typewriter effect
  typingTimeline.to(terminalText, {
    text: {
      value: terminalMessage,
      delimiter: ""
    },
    duration: 3,
    ease: 'none'
  });
  
  // Hide cursor while typing, show at end
  typingTimeline.to(cursor, { opacity: 0, duration: 0 }, 0);
  typingTimeline.to(cursor, { opacity: 1, duration: 0 }, 3);
});

// Reset button - clear text
resetBtn.addEventListener('click', () => {
  if (typingTimeline) typingTimeline.kill();
  gsap.to(terminalText, {
    text: { value: '' },
    duration: 0.3,
    ease: 'power2.out'
  });
  gsap.to(cursor, { opacity: 1, duration: 0 });
});

How It Works

GSAP's free TextPlugin allows you to animate text content character-by-character, creating classic typewriter and terminal effects.

TextPlugin Configuration

  • text.value - The text string to type
  • text.delimiter - Character to split by (empty string = character by character)
  • duration - Total time for the typing animation
  • ease - Timing function (use 'none' for linear typing)

Blinking Cursor

The cursor element uses a CSS @keyframes animation with step-end timing to create a sharp, authentic blinking effect.

TextPlugin is Free!

Unlike GSAP's SplitText (which requires a Club membership), TextPlugin is completely free and included with GSAP core!
