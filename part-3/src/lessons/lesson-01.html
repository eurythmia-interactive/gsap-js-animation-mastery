JavaScript

JavaScript 
import { gsap } from 'gsap';
import { splitText, unsplitText } from '../../utils/splitText.js';

const title = document.querySelector('#title');
const playBtn = document.querySelector('#playBtn');
const resetBtn = document.querySelector('#resetBtn');

// Split title into characters on page load
const chars = splitText(title, 'chars');

// Set initial hidden state
gsap.set(chars, {
  opacity: 0,
  y: 100,
  scale: 0,
  skewX: 45
});

// Play button - animate characters in with bounce
playBtn.addEventListener('click', () => {
  gsap.fromTo(chars,
    {
      opacity: 0,
      y: 100,
      scale: 0,
      skewX: 45
    },
    {
      opacity: 1,
      y: 0,
      scale: 1,
      skewX: 0,
      stagger: 0.05,
      ease: 'back.out(1.7)'
    }
  );
});

// Reset button - return to initial hidden state
resetBtn.addEventListener('click', () => {
  gsap.to(chars, {
    opacity: 0,
    y: 100,
    scale: 0,
    skewX: 45,
    stagger: 0.03,
    ease: 'power2.in'
  });
});

How It Works

We use a custom splitText() function to break text into individual character spans, then animate them with GSAP's stagger property for smooth cascade effects.

splitText() Function

  • splitText(element, 'chars') - Splits text into character spans
  • splitText(element, 'words') - Splits text into word spans
  • Returns array of <span> elements
  • Handles emoji and unicode correctly

Stagger Animation

The stagger property creates the cascade effect by animating each character with a delay. A stagger of 0.05 means each character starts 50ms after the previous one.

Performance Tip

Use will-change: transform on character spans to enable GPU acceleration. This makes text animations silky smooth.

Why Manual Split?

GSAP's SplitText is a paid plugin. Our custom splitText() function gives you the same control without requiring a license!
