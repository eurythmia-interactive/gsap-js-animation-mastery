Create looping animations that bounce back and forth

JavaScript

JavaScript 
import { gsap } from 'gsap';

// Infinite bounce with yoyo (back-and-forth)
gsap.to('#ball-1', {
  y: 250,
  duration: 0.5,
  ease: "bounce.out",
  repeat: -1, // -1 = infinite loop
  yoyo: true // Go back and forth
});

// Pulse effect (scale up and down)
gsap.to('#ball-2', {
  scale: 1.3,
  duration: 0.4,
  ease: "power2.inOut",
  repeat: -1,
  yoyo: true
});

How It Works

repeat and yoyo are two properties that work together to create looping animations.

Key Properties

repeat: n - Play the animation n+1 times
repeat: -1 - Loop forever
yoyo: true - Reverse direction on each repeat

How Repeat and Yoyo Work Together

Without yoyo:
[Start] → [End] → [Start] → [End] → [Start] → ...

With yoyo:
[Start] → [End] → [End] → [Start] → [Start] → [End] → ...

Repeat-Related Properties

Property Description Example
repeat Number of repeats -1 = infinite
yoyo Reverse on repeat true or false
repeatDelay Pause between repeats 0.5 = half second

Common Patterns

  • Bouncing ball: y: 300 with bounce.out
  • Pulsing: scale: 1.2 with power2.inOut
  • Shaking: x: '+=10' with power1.inOut
  • Fading pulse: opacity: 0.5 with power1.inOut
