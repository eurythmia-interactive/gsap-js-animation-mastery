JavaScript

JavaScript 
import { gsap } from 'gsap';

const box = document.querySelector('.demo-box');
const progressBar = document.querySelector('#progress-bar');

// Create paused tween (won't start automatically)
const tween = gsap.to(box, {
  x: 400,
  duration: 3,
  ease: "power2.out",
  paused: true, // Start paused!
  onUpdate: () => {
    progressBar.style.width = `${tween.progress() * 100}%`;
  }
});

// Control buttons
document.querySelector('#play-btn').onclick = () => tween.play();
document.querySelector('#pause-btn').onclick = () => tween.pause();
document.querySelector('#reverse-btn').onclick = () => tween.reverse();
document.querySelector('#restart-btn').onclick = () => tween.restart();

How It Works

GSAP tweens can be controlled after creation. Store a reference to your tween and call methods on it to control playback.

Key Control Methods

.play() - Start playing forward
.pause() - Pause the animation
.reverse() - Play backwards
.restart() - Start from the beginning

More Control Methods

Method Description
.seek(time) Jump to specific time
.progress() Get current progress (0-1)
.timeScale(n) Set playback speed (2 = 2x faster)
.kill() Stop and remove animation

The `paused: true` Option

By default, tweens start playing immediately. Use paused: true in the config to create a tween that waits for your command to start.

Use Cases

  • Play/pause buttons for animations
  • Video-style progress bars
  • Interactive storytelling (click to advance)
  • Game animations controlled by user input
