Stagger - Cascade Effects

Animate multiple elements with offset timing for wave effects

JavaScript

JavaScript 
import { gsap } from 'gsap';

// Select all boxes
const boxes = document.querySelectorAll('.demo-box');

// Animate all boxes with stagger effect
gsap.to(boxes, {
  x: 0,
  opacity: 1,
  duration: 0.5,
  ease: "power2.out",
  stagger: 0.1 // 0.1s delay between each box
});

How It Works

Stagger adds an offset delay between animations when targeting multiple elements. Instead of all elements animating at once, each starts after a set delay.

The Stagger Effect

Without stagger: [Box1] [Box2] [Box3] [Box4] [Box5]
All start at once

With stagger: [Box1] → [Box2] → [Box3] → [Box4] → [Box5]
Each starts 0.1s after the previous

Stagger Options

Option Description Example
stagger: 0.1 Fixed delay between each Simple cascade
stagger: { each: 0.1 } Explicit each option Same as above
stagger: { from: "end" } Which element starts first "start", "end", "center", "random"

The "from" Options

  • "start" - First element (default)
  • "end" - Last element starts first (reverse wave)
  • "center" - Middle element starts first
  • "random" - Random order

Pro Tip

Stagger works with gsap.to(), gsap.from(), and timelines! It's one of the easiest ways to create impressive visual effects.

Try It Yourself

Edit /src/lessons/lesson-08.js and experiment:

  • Faster stagger: Try stagger: 0.05 for quicker cascade
  • Slower stagger: Try stagger: 0.3 for dramatic effect
  • Reverse wave: Add stagger: { each: 0.1, from: "end" }
  • Different animations: Each box does something different using stagger
  • Random order: Use from: "random"
