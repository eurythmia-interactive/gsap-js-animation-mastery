JavaScript

JavaScript 
import { gsap } from 'gsap';

const box = document.querySelector('.demo-box');
const tl = gsap.timeline({ paused: true });

// Add animations and labels to mark sections
tl.to(box, { x: 200, duration: 1 })
  .addLabel('middle')
  .to(box, { scale: 1.5, duration: 1 })
  .addLabel('end')
  .to(box, { rotation: 360, duration: 1 });

// Button controls
document.querySelector('#btn-play').onclick = () => tl.play();
document.querySelector('#btn-start').onclick = () => tl.seek(0);
document.querySelector('#btn-middle').onclick = () => tl.seek('middle');
document.querySelector('#btn-end').onclick = () => tl.seek('end');

How It Works

Labels are named markers you place on a timeline. They help you organize animations into sections and navigate to specific points in time.

Key Methods

.addLabel('name') - Create a label at current position
.seek('labelName') - Jump to a specific label

Why Use Labels?

  • Organize code into named sections (like chapters)
  • Navigate to specific moments without calculating times
  • Debug timelines more easily
  • Create reusable animation sequences

Timeline Position Methods

Method Description
.addLabel(name) Add a named marker
.seek(position) Jump to time or label
.getLabelTime(name) Get time of a label
.currentLabel() Get current label name

Note

In this demo, we used paused: true on the timeline so animations don't play immediately. Click the buttons to navigate!
