JavaScript

JavaScript 
import { gsap } from 'gsap';

const card = document.querySelector('.demo-card');

// Hover animations
card.addEventListener('mouseenter', () => {
  gsap.to(card, {
    y: -15,
    scale: 1.08,
    boxShadow: "0 30px 50px rgba(255, 107, 53, 0.3)",
    duration: 0.4,
    ease: "power2.out"
  });
});

card.addEventListener('mouseleave', () => {
  gsap.to(card, {
    y: 0,
    scale: 1,
    boxShadow: "0 10px 20px rgba(0,0,0,0.1)",
    duration: 0.4,
    ease: "power2.out"
  });
});

// Click animation - rotates 360 degrees
card.addEventListener('click', () => {
  gsap.to(card, {
    rotation: '+=360',
    duration: 1.2,
    ease: "power3.out"
  });
});

How It Works

Animations can respond to user interaction by using standard DOM event listeners. GSAP tweens can be triggered on mouseenter, mouseleave, click, and more.

Mouse Events

mouseenter - Cursor enters element
mouseleave - Cursor leaves element
click - User clicks element

Relative Values

Notice the rotation: '+=15' - this uses GSAP's relative value syntax. It adds 15 degrees to the current rotation each click, rather than setting it to an absolute value.

Common Patterns

  • Hover effects: Animate on mouseenter, reverse on mouseleave
  • Click toggles: Check current state and animate accordingly
  • Drag: Use mousedown/mouseup for drag-like interactions

Important Notes

  • GSAP tweens on the same property will overwrite each other by default
  • Use overwrite: 'auto' or overwrite: true for automatic cleanup
  • Consider using gsap.quickTo() for high-frequency updates like drag
