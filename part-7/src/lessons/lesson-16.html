The Quiet Hour

by Sarah Jennings

The world holds its breath at dawn,

as shadows yield to light,

and every creature stirs anew.

In quiet moments, souls unfold,

like flowers meeting sun,

the heart discovers what it knows.

This is the gift of being here,

of seeing, feeling, loving,

the simple miracle of now.

JavaScript Animation
// Get all stanzas and lines
const stanzas = gsap.utils.toArray(".poem-stanza");
const lines = gsap.utils.toArray(".poem-line");

// Set initial states
gsap.set(stanzas, { opacity: 0, y: 30 });
gsap.set(".poem-title, .poem-author", { opacity: 0, y: -20 });

const tl = gsap.timeline();

// Title and author first
tl.to(".poem-title", {
  opacity: 1, y: 0, duration: 1, ease: "power2.out"
})
.to(".poem-author", {
  opacity: 1, y: 0, duration: 0.8, ease: "power2.out"
}, "-=0.5")
// Stanzas appear one by one
.to(stanzas, {
  opacity: 1,
  y: 0,
  stagger: 1.5,  // 1.5s between stanzas
  duration: 1,
  ease: "power2.out"
}, "+=0.5")
// Lines within each stanza fade in
.to(lines, {
  opacity: 1,
  stagger: 0.8,  // 800ms per line
  duration: 0.6,
  ease: "power1.out"
}, "-=2");
Key Concepts
// Poetic timing = slow, contemplative
// stagger: 1.5 for stanzas = reading pace
// Lines within stanzas are secondary reveal
// power1.out = gentle, not punchy
