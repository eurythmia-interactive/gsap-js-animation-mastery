Tagline Power

Corporate tagline with subtle draw-on effect

Live Demo - SVG Draw Animation
Excellence Through Innovation MERIDIAN PARTNERS
HTML Structure
<div class="tagline-container">
  <span id="tagline">Excellence Through Innovation</span>
  <svg class="tagline-underline">
    <line class="tagline-path" ...></line>
  </svg>
  <span id="company-name">MERIDIAN PARTNERS</span>
</div>
JavaScript Animation
// Register DrawSVG plugin
gsap.registerPlugin(DrawSVGPlugin);

// Split tagline into words using custom splitText
const splitTagline = splitText("#tagline", "words");

// Set initial states
gsap.set(splitTagline, { opacity: 0, y: 20 });
gsap.set(".tagline-path", { strokeDashoffset: 300 });

// Create timeline
const tl = gsap.timeline();

tl
  // Words fade in with stagger
  .to(splitTagline, {
    opacity: 1,
    y: 0,
    stagger: 0.15,
    duration: 0.6,
    ease: "power2.out"
  })
  // SVG line draws on
  .to(".tagline-path", {
    strokeDashoffset: 0,
    duration: 1,
    ease: "power2.inOut"
  }, "+=0.3")
  // Company name fades in
  .to("#company-name", {
    opacity: 1,
    letterSpacing: "0.3em",
    duration: 1,
    ease: "power2.out"
  }, "-=0.5");
Key Concepts
// DrawSVG animates SVG stroke properties
// stroke-dasharray = total length of path
// stroke-dashoffset = how much is hidden
// From 300 to 0 = complete draw on
