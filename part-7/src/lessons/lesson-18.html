Wildflowers

by Daniel Morrison

In fields where no one walks,

wildflowers learn to dance,

swaying to the wind's invisible song.

They ask for nothing, need no care,

just sun and rain and room to grow,

a quiet bloom that catches light.

Perhaps we too could learn to be

so simply, purely, gloriously

alive, like flowers in the wild.

JavaScript Animation
// Split title into words using custom splitText
const titleSplit = splitText("#poem-title", "words");
const stanzas = gsap.utils.toArray(".poem-stanza");

// Set initial states
gsap.set(titleSplit, { opacity: 0, y: 50 });
gsap.set(".poem-author", { opacity: 0, y: 20 });
gsap.set("#poem-divider", { scaleX: 0 });
gsap.set(stanzas, { opacity: 0, y: 40 });

const tl = gsap.timeline();

tl
  // Title word reveal with scale
  .to(titleSplit, {
    opacity: 1, y: 0,
    stagger: 0.15,
    duration: 0.8,
    ease: "back.out(1.5)"
  })
  // Author name
  .to(".poem-author", {
    opacity: 1, y: 0,
    duration: 0.6,
    ease: "power2.out"
  }, "-=0.3")
  // Divider draws across
  .to("#poem-divider", {
    scaleX: 1,
    duration: 1,
    ease: "power2.inOut"
  }, "+=0.2")
  // Stanzas fade in with stagger
  .to(stanzas, {
    opacity: 1, y: 0,
    stagger: 2,  // 2s between stanzas
    duration: 1.2,
    ease: "power2.out"
  }, "-=0.5");
Key Concepts
// Multiple animation types combined
// back.out for title's playful pop
// power2 for body content
// Divider draws using scaleX
// Stagger timing creates reading rhythm
