# GSAP + Tailwind Integration Course Ideas

## Course Overview

A collection of tutorial ideas combining GSAP's animation capabilities with Tailwind CSS's utility-first styling system.

---

## Lesson Ideas

### Part X: GSAP + Tailwind Integration Fundamentals

#### Lesson XX: Animated UI Cards with Tailwind + GSAP
**Concept:** Create interactive card components with Tailwind styling and GSAP hover effects.

- Cards with shadow and scale animations
- State class toggling for enhanced effects
