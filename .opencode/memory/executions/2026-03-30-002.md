## Key Learnings

### Combined Approach Philosophy
- **Tailwind handles:** colors, spacing, typography, basic transitions, layouts
- **GSAP handles:** complex timelines, scroll triggers, draggable, precise timing

### Integration Setup (Play CDN)
```html
<script src="https://cdn.tailwindcss.com"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/gsap.min.js"></script>
```

2. GSAP controls Tailwind state classes
3. Tailwind transitions + GSAP complex sequences
4. Responsive animation with Tailwind breakpoints
5. Dark mode aware animation
6. Tailwind transition + GSAP override
7. Animating Tailwind arbitrary values
